# 🚀 Современная CRM Система

Мощная и гибкая CRM-платформа, построенная на современном технологическом стеке. Основная особенность системы — **полная кастомизация данных** через конструктор сущностей, что позволяет адаптировать её под любой бизнес.

---

## ✨ Основные возможности

### 🏗️ Конструктор сущностей (Entity Builder)
Создавайте собственные типы данных прямо в интерфейсе:
- Добавление полей разных типов (текст, число, дата, выбор и др.).
- Настройка отображения и валидации.
- Динамическое создание форм для ввода данных.

### 👥 Мультитенантность и Компании
- Регистрация компаний как отдельных пространств.
- Изоляция данных между разными компаниями.
- Поддержка ролей: Владелец, Администратор, Менеджер.

### 🔐 Гибкое управление доступом
- Ролевая модель доступа (RBAC).
- Ограничение доступа к конкретным записям.
- **Access Expiration**: возможность выдавать временный доступ к данным с автоматическим отзывом прав.

### 🤖 Интеграция с Telegram
- Уведомления о важных событиях.
- Взаимодействие с системой через бота.
- Привязка Telegram-аккаунта к профилю пользователя.

### 📊 Дашборды и Аналитика
- Визуализация ключевых показателей (Recharts).
- Отслеживание активности и статистики по записям.

## 🛠 Технологический стек

### **Backend**
- **FastAPI** — высокопроизводительный веб-фреймворк на Python.
- **SQLAlchemy (Async)** — современная ORM для работы с БД.
- **PostgreSQL 16** — надежная реляционная база данных.
- **Alembic** — управление миграциями базы данных.
- **Redis** — кэширование, сессии и очереди задач.
- **APScheduler** — планировщик для фоновых задач (проверка истечения доступа).

### **Frontend**
- **React 18** + **TypeScript** — фронтенд-библиотека и строгая типизация.
- **Vite** — сверхбыстрый инструмент сборки.
- **TanStack Query (v5)** — управление состоянием серверных данных.
- **Zustand** — легкий и быстрый стейт-менеджер.
- **Tailwind CSS** — современный стилистический фреймворк.
- **React Hook Form** + **Zod** — работа с формами и валидация.

### **Bot**
- **Aiogram 3** — асинхронный фреймворк для Telegram-ботов.

## 🚀 Быстрый старт (через Docker)

Это самый простой способ запустить проект со всеми зависимостями.

1. **Клонируйте репозиторий и перейдите в него:**
   ```bash
   git clone <url-репозитория>
   cd crm
   ```

2. **Настройте переменные окружения:**
   ```bash
   cp .env.example .env
   # Отредактируйте .env: укажите SECRET_KEY, BOT_TOKEN и другие настройки
   ```

3. **Запустите систему:**
   ```bash
   docker-compose up --build
   ```

4. **Примените миграции (только при первом запуске):**
   ```bash
   docker-compose exec backend alembic upgrade head
   ```

Система будет доступна по адресам:
- **Frontend**: [http://localhost:3000](http://localhost:3000)
- **API Docs**: [http://localhost:8000/api/docs](http://localhost:8000/api/docs)

## 🛠 Ручная установка

### 1. База данных (PostgreSQL)
Создайте пользователя и базу данных:
```sql
CREATE USER crm WITH PASSWORD 'your_password';
CREATE DATABASE crm_db OWNER crm;
```

### 2. Backend
```bash
cd backend
python -m venv .venv
source .venv/bin/activate  # Или .venv\Scripts\activate для Windows
pip install -r requirements.txt
cp ../.env.example ../.env
# Настройте DATABASE_URL и другие параметры в .env
alembic upgrade head
uvicorn app.main:app --reload
```

### 3. Frontend
```bash
cd frontend
npm install
npm run dev
```

## 🔑 Первый вход

1. Перейдите на страницу регистрации: `http://localhost:3000/register`.

3. Теперь вы можете создавать сущности, добавлять пользователей и настраивать CRM под свои нужды.
